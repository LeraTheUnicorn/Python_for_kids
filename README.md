# Python для детей

Этот проект содержит примеры кода из книги "Python для детей. Самоучитель по программированию" Джейсона Бриггса.

## Описание книги

Эта книга позволит вам погрузиться в программирование и с легкостью освоить Python. Вы сможете написать несколько настоящих игр. На каждом шагу вы будете видеть результаты своих трудов — в виде работающей программы, а с понятными инструкциями и примерами с забавными иллюстрациями обучение будет только приятным.

## Требования

- Python 3.x
- Библиотеки: tkinter (обычно входит в стандартную установку Python), Pillow (PIL) для работы с изображениями

Установите Pillow, если необходимо:

```bash
pip install pillow
```

## Как запустить

Убедитесь, что у вас установлен Python 3 и необходимые библиотеки.

Для запуска любого файла выполните в терминале:

```bash
python chapter_3.py
```

Для reward.py:

```bash
python reward.py
```

Некоторые примеры открывают графические окна (tkinter), так что убедитесь, что ваша среда поддерживает GUI.
